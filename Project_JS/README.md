# Product Inventory System

This is a frontend web project built using HTML, CSS, and JavaScript.

## Features:
- Fetch product data from FakeStore API
- Product search
- Price filtering
- Sorting by price and name

## Technologies Used:
- HTML
- CSS
- JavaScript
- FakeStore API
